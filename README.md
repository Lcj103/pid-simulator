# PID 模拟器 — 一维小球位置控制

一个基于 Canvas 和纯 JavaScript 的 PID 控制器网页模拟器，用于演示在一维空间中对小球位置的控制。

## 功能特性

- **单级 PID 控制** — 位置环直接输出力
- **串级 PID 控制** — 外环位置环输出速度设定值，内环速度环输出力
- **实时可视化** — 小球运动动画、施力箭头、速度指示线
- **位置-时间响应曲线** — 15 秒滚动窗口，直观观察控制效果
- **可调参数** — Kp/Ki/Kd、积分限幅、输出限幅、质量、阻尼
- **仿真速度** — 支持 0.1x ~ 5x 倍速
- **鼠标交互** — 拖拽小球施加扰动，或直接拖动修改目标位置
- **外部扰动** — 点击按钮随机施加冲量，测试抗干扰能力
- **纯前端** — 零依赖，无需后端服务

## 使用方法

直接在浏览器中打开 `index.html`：

```
start index.html
```

### 操作说明

1. 点击 **启动** 开始仿真
2. 拖动 **目标位置** 滑块设定期望位置
3. 调节 PID 参数观察控制效果
4. 点击 **扰动** 模拟外界干扰
5. 切换到 **串级 PID** 模式体验双环控制

## 参数说明

| 参数 | 说明 |
|------|------|
| Kp | 比例系数，决定响应速度 |
| Ki | 积分系数，消除稳态误差 |
| Kd | 微分系数，抑制超调与震荡 |
| 阻尼 | 模拟摩擦阻力 |
| 质量 | 小球质量，影响惯性 |

## 技术栈

- HTML5 + CSS3
- Canvas 2D API
- 纯 JavaScript（ES6+）
- 欧拉法物理积分
- requestAnimationFrame 动画循环
